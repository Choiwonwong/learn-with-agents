# Go Track

Go 언어와 실용적인 백엔드/시스템 프로그래밍 감각을 학습하기 위한 트랙입니다.

## 예정 주제

- Go syntax
- Structs and interfaces
- Error handling
- Goroutines and channels
- Context
- HTTP server/client
- Testing

## 시작 규칙

Go 학습을 시작할 때는 작은 CLI 또는 HTTP 예제로 바로 실습 가능한 모듈을 만듭니다.

## 모듈

- [01. First CLI](basics/01-first-cli/README.md): GoLand로 `go.mod`, `package main`, 함수, struct, method, test, debug 흐름을 1시간 동안 맛봅니다.
