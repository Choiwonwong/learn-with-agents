# Fundamentals Track

언어와 무관하게 반복해서 쓰이는 컴퓨터 과학 및 엔지니어링 기본기를 다루는 트랙입니다.

## 예정 주제

- 자료구조
- 알고리즘 기초
- 타입과 추상화
- 네트워크 기초
- 파일과 I/O
- 테스트 사고
- 디버깅

## 시작 규칙

기본기 학습은 작은 구현과 설명을 함께 진행합니다. 문제 풀이만 하거나 개념 읽기만 하는 방식은 피합니다.
