# Learning

언어와 주제별 학습 트랙을 모아두는 공간입니다.

`learning/`은 단순 자료 보관소가 아닙니다. 각 모듈 안에서 직접 실습합니다.

## 트랙

- [Python](python/README.md)
- [Java](java/README.md)
- [Go](go/README.md)
- [Design Patterns](design-patterns/README.md)
- [Fundamentals](fundamentals/README.md)

## 권장 학습 루프

1. `guide.md`로 오늘의 목표를 확인한다.
2. `practice/`에서 직접 코드를 작성한다.
3. `reference/`와 비교한다.
4. `notes.md`에 회고를 남긴다.
5. 다음 실습 주제를 작게 정한다.

## 모듈 표준 구조

```text
learning/<track>/<area>/<NN-name>/
├── README.md
├── guide.md
├── practice/
├── reference/
├── notes.md
└── requirements.txt
```
