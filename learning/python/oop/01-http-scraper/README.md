# 01. HTTP Scraper

Python OOP를 작은 HTTP 스크래퍼로 익히는 2시간 실습 모듈입니다.

## 파일

- `guide.md`: 전체 학습 가이드북
- `practice/oop_scraper.py`: 직접 작성하는 실습 파일
- `reference/oop_scraper.py`: 비교용 완성 예시
- `notes.md`: 학습 회고와 질문 기록
- `requirements.txt`: 이 모듈에 필요한 외부 패키지

## 실행 준비

프로젝트 루트에서 가상환경을 켠 뒤 설치합니다.

```bash
python -m pip install -r learning/python/oop/01-http-scraper/requirements.txt
```

## 실행

직접 작성한 파일:

```bash
python learning/python/oop/01-http-scraper/practice/oop_scraper.py
```

비교용 완성 파일:

```bash
python learning/python/oop/01-http-scraper/reference/oop_scraper.py
```
