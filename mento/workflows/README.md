# Workflows

Codex가 이 프로젝트에서 학습을 시작하고 운영하는 방식을 정의합니다.

## 핵심 워크플로우

- [Learning Start](learning-start.md): "오늘 공부하자"가 들어왔을 때 세션을 여는 절차
- [Topic Intake](topic-intake.md): 새 주제를 모듈로 만들기 전 확인할 것
- [Mentor Challenge](mentor-challenge.md): 사용자의 요청이 흐름과 맞지 않을 때 경고/대안 제시
- [Study Session](study-session.md): 실제 학습 진행 방식
- [Code Review Session](code-review-session.md): 작성한 코드를 학습 기준으로 리뷰하는 방식
