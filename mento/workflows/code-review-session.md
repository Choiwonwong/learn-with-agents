# Code Review Session Workflow

사용자가 코드를 가져오거나 `practice/` 파일을 작성했을 때 사용합니다.

## 흐름

1. 코드가 어떤 학습 목표와 연결되는지 확인한다.
2. 실행 가능 여부를 확인한다.
3. `mento/rubrics/code-review-rubric.md` 기준으로 리뷰한다.
4. 가장 중요한 문제 1-3개만 먼저 고른다.
5. 다음 수정 하나를 지정한다.
6. 수정 후 다시 실행한다.

## 리뷰 우선순위

1. correctness
2. lesson fit
3. responsibility
4. simplicity
5. verification
6. Pythonic or language-idiomatic style
